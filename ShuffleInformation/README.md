ShufflyNode#
===========

An implementation of the Shuffly Concept using Node.js built in c# compiled with erik-kallen's SaltarelleCompiler

Changed Concepts
--------

* Games will be written in javascript
* Live debugging will be available

Dependencies
------------

* Nodejs v0.8.0+
* Fibers 
* socket.io
* MongoDB
* Redis
